# Retail Sales Intelligence Dashboard

## Overview

The Retail Sales Intelligence Dashboard is an end-to-end analytics project developed using Python, Prophet, and Power BI to analyze retail sales performance, customer behavior, product profitability, discount impact, and future sales trends.

The project converts raw retail transaction data into business-ready datasets and interactive dashboards that support data-driven decision-making.

## Business Problem

Retail businesses need to monitor sales performance, profitability, customer behavior, and product performance to make informed business decisions. However, analyzing large volumes of transactional data manually is time-consuming and inefficient.

This project provides an interactive business intelligence solution that enables users to track key performance indicators, evaluate business trends, and forecast future sales performance.

## Project Workflow

### Data Collection

Used the Superstore retail dataset containing transaction-level sales, customer, product, and profitability information.

### Data Cleaning & Preparation

* Cleaned and transformed raw sales data
* Standardized data formats
* Created business-ready datasets
* Generated separate datasets for customer, product, and forecasting analysis

### Sales Forecasting

Used Prophet to forecast future sales trends based on historical sales data.

### Dashboard Development

Built an interactive Power BI dashboard to analyze sales performance, profitability, customer behavior, product trends, discount impact, and sales forecasts.
